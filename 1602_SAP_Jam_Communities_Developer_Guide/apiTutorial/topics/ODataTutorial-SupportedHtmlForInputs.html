Supported HTML for Inputs

This page lists the HTML tags that are supported in various SAP Jam OData API inputs such as inputs for wikis, blogs, Questions, Ideas, and Discussions.

Basic HTML inputs can be either escaped or wrapped using a CDATA tag. Not all HTML is supported, for example scripts, frames and iframes, and forms are not supported, but the following tags, listed by category, are allowed:

Note: All HTML input is sanitized; only valid HTML will be accepted.
