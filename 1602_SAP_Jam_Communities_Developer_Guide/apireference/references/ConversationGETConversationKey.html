[GET] /Conversations('Id')

Retrieves the information on the specified Conversation.

Request Elements

URL

https://<your_company>.<communities#>.sapjam.com/api/v1/OData/Conversations('Id')

HTTP Request Method

GET

CURL Commands

XML

curl -H "Authorization: OAuth <!--[The OAuth1.0a protocol parameters have been removed from here.]-->"
     -H "Accept: application/xml" -H "Content-Type: application/xml"
     "https://<your_company>.<communities#>.sapjam.com/api/v1/OData/Conversations('fg7Iv7gFvm9rAW6m7fxBLn')"

JSON

curl -H "Authorization: Bearer <!--[An OAuth2.0 auth_token has been removed from here.]-->"
     -H "Accept: application/json" -H "Content-Type: application/json"
     "https://<your_company>.<communities#>.sapjam.com/api/v1/OData/Conversations('fg7Iv7gFvm9rAW6m7fxBLn')"

HTTP Input Headers

User access tokens must be passed in the API call. This can be done using OAuth1.0a or OAuth2.0. For OAuth1.0a we recommend using either the HMAC-SHA1 or RSA-SHA1 signature methods, but not PLAINTEXT. Also, OAuth1.0a authentication can be passed in three different ways, but passing it in the header is the preferred method, and it is the method shown here.

Please refer to the Authorization API section for more information.

Generic Example:

URL Parameters

URL Segment Required Description
Id Mandatory The unique ID of the Conversation for which you want to retrieve information.
access_token Mandatory for POST, PATCH, and DELETE calls; optional for GET calls. The current logged-in user's SAP Jam Communities access token. For development purposes, use the access-token generated as described in the Communities API Authorization page.

Conversation GET and POST Response Payload Properties

Note: The property "type" only appears in XML responses.
Property Type Description
Id Edm.String The unique ID of the Conversation.
Title Edm.String The title given to the Conversation by its creator.
HighlightedTitle Edm.String The title set if the Conversation was highlighted.
OriginalContent Edm.String The first post that the creator of the Conversation wrote.
HtmlContent Edm.String The HTML content of the Conversation.
TypeTrait Edm.String The type of the Conversation.
IsFeatured Edm.Boolean True if the Conversation is featured.
HasFeatured Edm.Boolean True if the Conversation was ever featured.
LikeCount Edm.Int32 A count of the "likes" that the Conversation has received.
FollowCount Edm.Int32 A count of the number of members who are following the Conversation.
ReplyCount Edm.Int32 A count of the number of replies that the Conversation has received.
ViewCount Edm.Int32 A count of the number of views that the Conversation has received.
VoteCount Edm.Int32 A count of the number of votes that the Conversation has received.
IsClosed Edm.Boolean True if the Conversation has been closed by a moderator.
CreatedAt Edm.DateTimeOffset The date and time that the Conversation was created.
UpdatedAt Edm.DateTimeOffset The date and time that the Conversation was last modified.
RootPostPreview Edm.String The preview content of the initial post of the Conversation.
TopicTitles Edm.String The titles (tags) that have been set for the Conversation.
PrettyTitle Edm.String The title set by a moderator for the Conversation.
RootPostId Edm.String The unique ID of the initial post of the Conversation.
