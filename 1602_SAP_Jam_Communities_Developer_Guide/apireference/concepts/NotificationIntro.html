OData Aggregation API calls

This section documents SAP Jam Communities OData API calls that involve Aggregations.

The Aggregation entity

Purpose: The Aggregation entity provides information on .

Properties: The Aggregation properties are:

  • Id: The unique ID of the task.
  • Description: The content, or message, of the Task.
  • CreatedAt: The date and time that the Task was created.
  • Title: The name of the Task.
  • LastModifiedAt: The date and time that the Task was last modified.
  • DueAt: The date and time by which the Task must be completed.
  • Priority: The priority that the Creator sets for the Task. The valid values are "low", "medium", or "high".
  • Reminder: The frequency of email reminders to the Assignee. The valid values are "none", "daily", "weekly", or "monthly".

Navigations: The Tasks entity provides navigations to:

  • Assignments: For information on the TaskAssignments associated with the specified Task.
  • Attachments: For information on the Attachments that have been added to the specified Task.
  • Creator: For information on the Member who created the specified Task.
  • Event: For information on the meeting Event associated with the specified Task.
  • FeedEntries: For information on the FeedEntries added as discussion messages to the specified Task.
  • Followers: For information on the Members who are Followers of the specified Task.
  • Group: For information on the Group in which the specified Task is located.
  • PendingFollowers: For information on the Members who have been invited to follow the specified Task, but who have not yet responded.
