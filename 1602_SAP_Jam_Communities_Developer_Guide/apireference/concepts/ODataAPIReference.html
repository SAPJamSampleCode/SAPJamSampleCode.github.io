SAP Jam Communities OData API Reference

SAP Jam Communities offers OData API calls that allow you to integrate SAP Jam Communities features into your business critical applications, and data from your business critical applications into SAP Jam Communities. This particular page provides a graphical overview of the SAP Jam Communities OData API entity hierarchy and interactions, as well as a list of the API endpoints. Subsections introduce more detailed information on each entity or set of related entities, and pages below those subsections provide the details on each API call or endpoint.
  • Conversation API calls
    This section documents SAP Jam Communities SAP Jam Communities API calls that involve Conversations.
  • Post API calls
    This section documents SAP Jam Communities SAP Jam Communities API calls that involve Posts.
  • Topic API calls
    This section documents SAP Jam Communities SAP Jam Communities API calls that involve Topics. Topics are typically "products" on an e-commerce site, but they could also be "courses" in a Learning Management System site. Topics are the main, specific object of interest and discussion.
