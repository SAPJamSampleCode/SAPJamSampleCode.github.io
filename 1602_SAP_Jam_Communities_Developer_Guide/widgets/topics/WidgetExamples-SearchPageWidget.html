Search page widget

The search page widget, shown in the following screen capture, displays search results and faceted filters. This widget offers sorting and pagination functionalities.

Figure 1. The search page widget

To display the search page widget for a specific product, you must first specify the search string that you want to display, as is shown in the following code example. You must also specify the search keyword, the parent div ID in which you want both the facet search options and the keyword search results displayed, and you must specify the name of the widget type to be displayed.

Excelsior.ready(function() {
    var keyword = location.search.match(/((text)=(.+?)((%3A.)|$)/)[4];

    Excelsior.setSearchKeyword(keyword);
    Excelsior.render({
        id: ['facetNavigation','ex-searchResult'],
        name: 'SearchCommunity'
    });
}
