The home page widget

The home page widget, shown in the following screen capture, displays a variety of content related to a specific product.

Figure 1. The home page widget

To display the home page widget for a specific product, you must first specify the product code of the product that you want to display, as is shown in the following code example.

Excelsior.ready(function() {
    Excelsior.setProductCode('1934793');   // specified product
}

You must also specify the parent div ID in which you want the home page widget displayed, and you must specify the name of the widget type to be displayed.

Excelsior.ready(function() {
    Excelsior.render({
        id: 'con3',    // parent div ID 
        name:'HomePageReview'
        });
}
