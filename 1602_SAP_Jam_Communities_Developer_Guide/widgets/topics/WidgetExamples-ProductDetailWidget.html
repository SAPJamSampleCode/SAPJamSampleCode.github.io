The product detail widget

The product detail widget can display either the most recent review of a specific product or the most recent question about the product.

The simple review option

The product detail widget, shown in the following screen capture, displays the most recent review for a specific product, as well as the product's aggregated rating score.

Figure 1. The product detail widget (SimpleReview option)

To display the product detail widget's simple review option for a specific product, you must first specify the product code of the product that you want to display, as is shown in the following code example.

Excelsior.ready(function() {
    Excelsior.setProductCode('1934793');   // specified product
}

You must also specify the parent div ID in which you want the product detail widget's simple review option displayed, and you must specify the name of the widget type to be displayed.

Excelsior.ready(function() {
    Excelsior.render({
        id: 'con3',    // parent div ID 
        name:'SimpleReview'
        });
}
