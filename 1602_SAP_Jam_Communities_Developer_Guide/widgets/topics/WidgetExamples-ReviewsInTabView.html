Review tab view

The review tab view widget, shown in the following screen capture, displays a page of reviews for a specific product, offers aggregated rating information, and provides sorting and pagination functionality.

Figure 1. The review tab view widget

To display the reviews tab view widget for a specific product, you must first specify the product code of the product that you want to display, as is shown in the following code example.

Excelsior.ready(function() {
    Excelsior.setProductCode('1934793');   // specified product
}
Excelsior.ready(function() {
    Excelsior.render({
        id: 'con3',    // parent div ID 
        name:'ReviewTab'
        });
}
