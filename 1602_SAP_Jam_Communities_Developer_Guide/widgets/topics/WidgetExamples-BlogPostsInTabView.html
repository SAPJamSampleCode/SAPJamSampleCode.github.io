Blog posts tab view

The blog post tab view widget displays a page of blogs about a specific product. This widget offers sorting and pagination functionalities.

To display the blog posts tab view widget for a specific product, you must first specify the product code of the product that you want to display, as is shown in the following code example.

Excelsior.ready(function() {
    Excelsior.setProductCode('1934793');   // specified product
}

You must also specify the parent div ID in which you want the blog posts tab page widget displayed, and you must specify the name of the widget type to be displayed.

Excelsior.ready(function() {
    Excelsior.render({
        id: 'con3',    // parent div ID 
        name:'BlogTab'
        });
}
