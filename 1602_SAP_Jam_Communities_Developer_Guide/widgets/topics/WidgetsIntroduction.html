The SAP Jam Communities UI Widget Library

SAP Jam Communities offers a JavaScript library which makes it possible to easily embed a number of different widgets that can display your organization's SAP Jam Communities content and functionality in a variety of different contexts or experiences. This document provides instructions on how to:

  • Include the SAP Jam Communities UI widgets library in a web page.
  • Specify the widget or view that you want to display.
  • Specify the product that you want the widget to display information about.

Assumptions

This document assumes you have access to an instance of SAP Jam Communities, and that this instance has been properly integrated and synchronized with another system, or systems, that serve as the source or master for your product catalog information. Currently, this integration has only been verified with SAP hybris Commerce. While integrations with other e-commerce applications, and even other types of web application, are theoretically possible, they have not been tested and will not be supported by SAP without prior approval. For further information on this topic, please refer to the SAP Jam Communities Administration Guide.

Prerequisites

In order to successfully embed an SAP Jam Communities UI widget in a web page you will need to be able to access the SAP Jam Communities Widget Library. There are 2 way of enabling this:

  • You can download the SAP Jam Communities UI widget library to your organization's local system and set the statements required to include it in your organization's web pages. This approach is not recommended as it breaks SAP's ability to perform ongoing feature updates, as well as any required issue fixes or security improvements. As a result, SAP will not support or provide assistance in using this option.
  • You can include the SAP Jam Communities UI widget library from SAP's cloud-based content distribution location. This is the recommended approach for all SAP Jam Communities UI widget library integrations.
