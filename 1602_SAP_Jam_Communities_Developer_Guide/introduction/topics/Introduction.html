The SAP Jam Communities Developer Guide

This document, the SAP Jam Communities Developer Guide, presents developer-oriented information for integrating SAP Jam Communities into web pages using the SAP Jam Communities UI Widget Library or into full web applications, such as e-commerce sites, using the SAP Jam Communities OData API.

The differences between these two approaches are as follows:

  • SAP Jam Communities UI Widget Library: provides access to your organization's existing SAP Jam Communities content via a JavaScript widget, although minimal restructuring of that content is possible.

    These widgets include:

    • Home page widget: displays a variety of content on a specific topic, which is typically related to a specific product.
    • Product detail widget: displays the most recent question or review posted about a specific product (topic). Several different views are available:
      • Reviews in tab view: displays a page of the reviews for a specific product.
      • Questions in tab view: displays a page of the questions and answers for a specific product.
      • Blog posts in tab view: displays a page of the blogs for a specific product.
      • Discussions in tab view: displays a page of the discussions for a specific product.
    • Search page widget: displays the results of a search with access to faceted filters that can be toggled on and off to refine the list of results that are displayed.
  • SAP Jam Communities OData API: provides access to your organization's existing SAP Jam Communities content by developing page structures for that content that pull in SAP Jam Communities data using the API. This approach allows maximum flexibility in the structuring of that content into your existing web applications.

    The API calls allow you to show:

    • Topics: provides the data to display a variety of content related to a specific topic, which is typically a specific product.
    • Conversations: provides the data to display the content of specific conversations, whether they are in response to reviews, questions, blog posts, or discussions.
    • Posts: provides the data to display content related to specific posts.

SAP Jam Communities Developer Guide content

This guide is organized into four "tabs", which provide the following information:

  • Introduction: provides this introductory page, as well as SAP's legal notices and copyright information.
  • Widgets: provides instructions on how to use the SAP Jam Communities UI Widget Library to add SAP Jam Communities into web pages.
  • API Tutorial: provides information on how to use the SAP Jam Communities OData API.
  • API Reference: provides a reference on each API call available in the SAP Jam Communities OData API.
